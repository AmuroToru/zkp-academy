前两部分的课程形式是周一到周五学员根据材料学习，期间可以自由组织讨论会，周六老师们会分享过一下学习材料并答疑。

另外可以增加课程中相关的分享，比如饭卡老师准备讲一下一个stark怎么写，欢迎各位老师报名。
